HShare is a website for students to learn and teach each other regarding various educational questions. Students can ask questions they are confused in and other students can share the answers and help solve it.

  1. Students can ask , answer and discuss confusing questions
  2. Students can upload answers (pdf, pptx, docx, xlsx, txt)
  3. If you have any problems , you can easily contact the admins
  4. Features will be constantly added (it is still under construction )

How Does HShare Work ?

HShare has a system of servers and channels. Each Server can represent your class and each channel can represent your subject. There can be public server for everyone to use and you can also make your private server with password protection.If you want a private server please contact the admins.
