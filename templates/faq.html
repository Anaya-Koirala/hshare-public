1. Is HShare free and can anyone use it ?

Yes , HShare is free to use with no hidden cost.It is made by students for students to help each other in educational field.

2. How does HShare work , Exactly ?

HShare has a system of servers and channels.Each server can be a subject ( for example Maths) and each server in this case Maths has various sub groups called channels.Each channel can represent a grade . In case of private servers , The server can represent your school and channels the subject.
You can learn more at here

3. What is a private server ?

A private server is a type of server that is created for your group/school.This can be created by contacting the admins. Your private server can also be password protected for privacy but the owners can access it too.Each private server can represent your school and channel the subject taught.
If you want a private server please contact the admins here

4. What can be uploaded and what can't in posts

You can just upload description as answer or can upload the images here if you want to get fancy.
Unfortunately for now the file types supported are : .pdf , .pptx , .docs , .xslx , .txt
Support for images is comming soon
